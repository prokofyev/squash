# 🏓 Squash Game

Классическая аркадная игра "Сквош" для двух игроков на Python с использованием Pygame.

## 📖 Описание

Два игрока соревнуются в отбивании мяча, не давая ему упасть вниз. Игра ведётся до 5 очков. Мяч отскакивает от верхней стенки экрана, а игроки по очереди должны отбивать его своими ракетками. Если игрок не успевает ударить по мячу в свой ход, очко засчитывается сопернику.

## ✨ Особенности

- **Цветовая индикация хода** — полоса вверху экрана показывает, чей сейчас ход
- **Управление углом отскока** — мяч летит в зависимости от того, в какую часть ракетки он попал
- **Мяч появляется сверху** — каждый раунд начинается с подачи из верхней части экрана

## 📋 Требования

- Python 3.10 или выше
- Pygame

## 🎯 Управление
|Игрок|Влево|Вправо|
|Игрок 1 (Синий)|← (стрелка влево)|→ (стрелка вправо)|
|Игрок 2 (Красный)|A|D|
Перезапуск после победы: R

## 📜 Правила игры
1. Игроки отбивают мяч по очереди
2. Если мяч проходит мимо ракетки в чей-то ход — соперник получает очко
3. Если не ваш ход — мяч пролетает сквозь вашу ракетку
4. Мяч отскакивает от боковых стенок и верха экрана
5. Удар по краю ракетки изменяет угол отскока мяча
6. Игра заканчивается при наборе 5 очков одним из игроков

## 💡 Советы по игре
- Бейте мяч краем ракетки, чтобы изменить его траекторию
- Следите за цветной полосой вверху — она показывает, чей сейчас ход
- Не стойте на месте — предугадывайте направление полёта мяча
- Центр ракетки отправляет мяч почти вертикально, края — под острым углом
